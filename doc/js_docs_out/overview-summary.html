WireIt

This document is the API Specification for WireIt.

Summary

WireIt is a library that allows you to create "wired" applications.

It is strongly inspired from Yahoo! Pipes, but aims to widen the use of "wired" capabilities.

It is built on top of the Yahoo! User Interface library. (version 2.3.1)

File Summary
Anim.js Provide a wrapper around YAHOO.util.Anim to animate a block containing terminals and redraw the associated wires
BoxIO.js Contains 2 helpful classes to create Input/Output boxes
DD.js Provide a wrapper around YAHOO.util.DD to drag/drop a block containing terminals and redraw the associated wires
Terminal.js Terminals represent the end points of the "wires"
Wire.js Contains the class WireIt.Wire that wraps a canvas tag to create wires
WireIt.js Set the WireIt namespace ; DOM and Canvas helpers
