About

   WireIt is an open-source javascript library, that allows you to create cool wires like Yahoo Pipes. You can see it in action in the Planar Game, or in the examples below.

   It currently supports Firefox 2, Safari 2, IE 7 and Opera. Please report your issues with specific browsers in the forum.

   It uses the YUI library for DOM and events manipulation, and excanvas for IE support.

Documentation

   WireIt comes with an exhaustive API documentation built with JSDoc,
and the following examples :

  • Creating terminals and wires
  • Move and Animate elements containing terminals
  • Changing terminal direction
  • Using BoxIO
